# GitHub Branching Sandbox

This is a deliberately tiny fake project for practicing Git and GitHub workflows in Codex.

Repository: https://github.com/dwagnon/chatgpttesting

The project pretends to be the homepage copy for a cafe. The files are plain text on purpose:

- `menu.txt` is for normal feature branch changes.
- `homepage.txt` is for the merge conflict exercise.
- `PRACTICE.md` is the guided walkthrough.

Nothing here is important. Break it, fix it, reset it, and try again.
